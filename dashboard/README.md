# SODCET Dashboard

An interactive React dashboard for analysing the categorisation of open data portals in Germany. It visualises the aggregated categories per federal state and shows the most and least used categories using Chart.js.

## 📊 Features

- Bar chart of all used categories
- Doughnut chart of the most frequently used categories
- Bar chart of the rarest categories
- Database: `combined_statistics.json` generated by SODCET

## 🚀 Setup

### Prerequisites

- Node.js ≥ 18
- npm ≥ 9

### Installation

```bash

```

Then replace the content of `src/App.jsx` with your `Dashboard.jsx` component.

### Install Chart.js

```bash
npm install chart.js react-chartjs-2
```

### Add data

Copy your `combined_statistics.json` file to:

```
dashboard/public/output/combined_statistics.json
```

### Start

```bash
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in the browser.

## 🧠 Source

The data originates from the project **SODCET - Semantic Open Data Category Extraction Tool** developed at Chemnitz University of Technology, Chair of Data Management.
